CSS-Doodle

CSS-Doodle 使用 CSS grid 的特性，可以在一個畫布上，為每個單元格設計出隨機且複雜的數學運算是背景或動畫

選擇器

:doodle(容器)

  1. :doodle{} 內，為容器的布局樣式
  2. :doodle{} 外，為每個單元格的樣式
  3. 以下為繪製 5 x 5 / 500px 的 grid，gap 為格子間的距離
:doodle{ @grid:5x5/500px; gap:10px } background:#FFA

:container

:doodle{ } :container{ @grid:5x5/500px; gap:10px } background:#777; border:3px solid #000; transform:rotate(@rand(360deg));

@nth(n, ...)

像 :nth-child(n) 選擇第 n 個單元格

:doodle{ @grid:5x5/500px; gap:10px } background:#FFA; @nth(odd){ background:green }

@even 與 @odd

:doodle{ @grid:5x5/500px; gap:10px } background:#FFA; @odd{ background:green }

@at(row, col)

指定第幾列 / 第幾欄的單元格

:doodle{ @grid:5x5/500px; gap:10px } background:#FFA; @at(4,3){ background:green }

@random([ ratio ])

隨機選擇單元格，@random 沒有比率時可以省略括號

使用比率，ratio 比率介於 0 ~ 1 之間，預設值為 0.5。

:doodle{ @grid:20x20/500px; } @random { border-top: 1px solid #60569e; } @random { border-left: 1px solid #60569e; } @random(.2) { :after { content: ''; background: hsl(@rand(360), 60%, 70%); @size: @rand(3px); } }

@row(n, ...)

璇則往格中的第 n 列

可使用 odd 與 even 選擇

:doodle{ @grid:5x5/500px; gap:10px } background:#FFA; @row(3){ background:green }

@col(n, ...)

璇則往格中的第 n 欄

可使用 odd 與 even 選擇

:doodle{ @grid:5x5/500px; gap:10px } background:#FFA; @col(odd){ background:green }

屬性

@place-cell

相當於 :doodle 容易來定位單元格。第一個數值為水平，第二個數值為垂直，可以使用 left、 right、 center、 top、 bottom 或是絕對數值 px ...等等，就像使用了 position:absolute 去調整位置

:doodle{ @grid:5x5/500px; gap:10px; background:#eee; margin:auto; } background:#ffa; @size:3rem; @nth(1){ @place-cell:0 top; background:red; z-index:1; } @nth(2){ @place-cell:right 25%; background:green; }

@shape

:doodle{ @grid:5x5/500px; gap:10px; background:#eee; margin:auto; @shape:heart; } background:#FFA; @shape:hypocycloid, 5

try

:doodle{ @grid: 5x1 / 500px; margin:auto } @place-cell: center; @size: calc(100% / 5 * @index); background:black; opacity: calc(1.1 - 1 / 5 * @index);
